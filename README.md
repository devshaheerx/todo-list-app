# To-Do List App

A lightweight and responsive To-Do List web app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add new todos by typing and pressing Enter or clicking the + icon
- Edit existing todos with a prompt dialog
- Delete todos instantly
- Todos persist after page refresh using localStorage
- Smooth hover animations on list items and icons

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- Font Awesome (icons)
- Google Fonts (Inter)

## Getting Started

1. Clone the repository
   git clone https://github.com/devshaheerx/todo-list-app.git

2. Open index.html in your browser — no install or build step needed.
